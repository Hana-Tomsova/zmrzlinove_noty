# Zmrzlinove noty

Detska online hra na procvicovani not hudebni abecedy.

Hraci prirazuji barevne kopecky zmrzliny ke kornoutum s pismeny:

- C - cerveny kopecek
- D - oranzovy kopecek
- E - zluty kopecek
- F - svetle zeleny kopecek
- G - zelenomodry kopecek
- A - fialovy kopecek
- H - ruzovy kopecek

Hra funguje kliknutim i pretazenim kopecku na spravny kornout.
